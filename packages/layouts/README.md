# Layouts-rc
Component library for React layouts.
## install
```npm install layouts-rc```
## style apply

```typescript jsx
// import './globals.css';
import 'layouts-rc/styles.css';
```
## usage
```typescript jsx
import {
    Scaffold,
    ScaffoldBody,
    ScaffoldFooter,
    ScaffoldHeader,
    Row,
    Column,
    Center
} from 'layouts-rc';
```
